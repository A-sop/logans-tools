# logans-tools — Add remote and push

**1. Create the GitHub repo first** (if not done):
- Go to https://github.com/new
- Repository name: `logans-tools`
- Leave empty (no README, .gitignore, license)
- Create

**2. Add remote and push:**
```powershell
cd c:\Dev\logans-tools
git remote add origin https://github.com/A-sop/logans-tools.git
git push -u origin master
```

If your GitHub username/org differs, replace `A-sop` with yours in the URL.
